Import the project in Netbeans or Eclipse.

Run the File EasyNOTE.java

Don't forget to add Tabs.java file

Salient Features:

New File
Open
Save, Save As, Exit (with saving notification)
New Tab
Undo, Redo (Using Clipboard), Cut, Copy, Paste
Find and Replace
Spell Check (with Suggestions) 
Font -> Color, Size, type, Style (Bold, Italics, Underline) 
UI-> Line Number on the left pane, Frame, Icons
Status Bar with Word Count, Current Cursor Position, Total Character Count
